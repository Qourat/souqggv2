# Agent: Product Creation

**Trigger**: admin selects an idea and clicks "Draft product".

## Output
A draft of the actual product (PDF guide, Excel structure, prompt pack, or
script bundle) suitable for upload after admin review.

## Rules
- No copying of copyrighted material — paraphrase always with attribution to
  the public-domain source if any.
- Arabic-first wording; English provided as a parallel translation when
  appropriate, never as the primary draft.
- Practical content: instructions, checklists, examples. No theory padding.
- No fake claims (income guarantees, miracle results).
- For Excel agents: include tabs, headers, formulas, sample row, instructions.
- For prompt packs: 20–50 prompts, grouped by use-case, with example outputs.
