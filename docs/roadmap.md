# souq.gg — Roadmap

## V1 (MVP — current)
Admin uploads products. Buyers browse, buy, download. Internal AI tools for
admin only. Stripe primary, RTL Arabic-first, Postgres FTS.

## Phase 1.5 (post-launch)
- PayTabs / Telr adapter for MENA card flows
- Bundle composer (multiple products as one purchasable unit with discount)
- Email automations (abandoned cart, post-purchase 7-day follow-up)
- Cmd-K command palette across the app
- Light theme polish

## Phase 2 (do NOT build during MVP)
- Seller onboarding & approval flow
- Seller dashboard, sales analytics, payout requests
- Marketplace submissions queue with admin review
- PDF watermarking pipeline (buyer email + order id)
- Meilisearch / Typesense for richer search
- Affiliate / referral program

## Phase 3
- Mobile app (Expo + Supabase)
- Subscription / membership tier
- Public reviews with moderation queue at scale
- Multi-currency display with FX
- API for third-party syncing
